### 描述
计算逆波兰式（后缀表达式）的值
运算符仅包含"+","-","*"和"/"，被操作数可能是整数或其他表达式
例如：

>["20", "10", "+", "30", "*"] -> ((20 + 10) * 30) -> 900
>["40", "130", "50", "/", "+"] -> (40 + (130 / 50)) -> 42

### 示例1
```
输入：["20","10","+","30","*"]
返回值：900
```
